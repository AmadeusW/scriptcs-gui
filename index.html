scriptcs-gui

A simple GUI toolkit for ScriptCS

The following script shows you how to read a folder name from the command argument and use it with the Gui.OpenFileDialog helper.

#load "../script/gui.csx"

var folder = ScriptArgs.FirstOrDefault(); 
var path = Gui.OpenFileDialog(folder);
Console.WriteLine("You selected: '{0}'.", path ?? "nothing");

Launch the script as shown in the picture and enjoy your interaction... :)

Other things you can do with ScriptCS-GUI:

  • Open a PopUp Windows Form from the console and programmatically add controls to it while both the console and the form are responsive!
  • Open a Drawing Windows Form from the console and draw on it from the console using classic Windows.Drawing commands.
  • More to come... :)
